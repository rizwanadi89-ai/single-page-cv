# Single-Page CV

A single-page HTML CV project built for [roadmap.sh](https://roadmap.sh/projects/single-page-cv).

## Live Preview

https://rizwanadi89-ai.github.io/single-page-cv

## About

This project is a single-page CV (Curriculum Vitae) built using only HTML, featuring:
- Semantic HTML structure
- SEO meta tags
- Open Graph (OG) tags
- Favicon
- Sections for Skills, Education, and Experience
